# 事件采集表

请尽量提供真实、可核对、时间明确的事件。事件越具体，候选时辰比较越有意义。

| 日期 | 事件 | 类型 | 确定度 | 可验证材料 | 备注 |
|---|---|---|---|---|---|
|  |  | 家庭/学业/事业/迁移/感情/健康/官非/财务 | 高/中/低 |  |  |

## 建议事件类型

- 家庭：父母变化、搬家、亲属重大事件。
- 学业/事业：升学、毕业、入职、离职、升迁、创业。
- 迁移：异地、出国、长期旅行、城市变更。
- 感情：确立关系、分手、结婚、离婚。
- 健康：手术、住院、长期不适；只用于时间校验，不作医学诊断。
- 官非/财务：合同纠纷、处罚、重大收入支出；现实问题仍需专业人士。

## 记录要求

不要只给“那年不顺”这类模糊描述；请写事件发生年月、影响程度和确定度。
