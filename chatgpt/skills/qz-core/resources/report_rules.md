# Core 报告规则

Core 是专题 Skill 的上游，报告要便于继承和复核。

## 必备顺序

1. 数据完整性复核：说明哪些字段可用，哪些字段低可信度。
2. 太极与命身：确认命宫、命度、命度主、身宫、身度、身度主和性别取法；并按命型（木火土月日金水）读出本命之性、衡量命主星旺衰，作为成全本性的起手判断（见 wuxing_per_ming.md）。
3. 尽三起手（只论宫）：从月令和本命之性出发，三步且每步只看本宫及三方四正——第一步命宫、第二步命主星所在宫、第三步月亮（身）所在宫，先定喜忌骨架与用神。
4. 三度论度断：尽三论宫之后再论度，按命度、命主星所在度、身度三度，各看其四度相通（同一五行四度互通）上的星；度为天，用来救、成或破尽三，宫度两条线不混看。
5. 原局结构：在尽三和度层骨架下看星曜、神煞和宫位专题，不以单星定论。
6. 五行体用喜忌：说明为什么喜、为什么忌，不能机械套生克。
7. 运势引动：先看原局有没有信息，再看流年、限度、倒限是否触发。断流年按命→限→流顺序：大限分宫限与度限，标克限/犯限；流年先顶星、无则太岁填冲，填吊星一作用命二作用限且不可倒回；多星并象、各自取象；倒限须限全伤且无生机（详见 transit_year_rules.md）。
8. 可继承结论：写成专题可复核的证据点，而不是绝对断语。

## 尽三位置

尽三不是 Reader 的职责，也不是专题 Skill 可跳过的步骤。它属于 qz-core 的正式解盘起手：Reader 先把命身、宫度、星曜提取完整；Core 定太极后立即进入尽三。尽三只论宫，三步分别看命宫、命主星所在宫、月亮所在宫的本宫及三方四正。教材明确提示“前面尽三全部是论宫组合，那么接下来论度……论宫当论宫，论度当论度”，所以尽三之后才论度，按命度、命主星所在度、身度三度各看四度相通；度层用于救、成或破尽三结论，宫度不可混看，然后才继续看星曜、神煞、五行体用和运势。

## 正反双审

每个核心判断必须列支持因素、制约因素、缺失数据和不确定项。若支持证据少于两层，应降级为“可观察信号”。

## 禁止写法

不写恐吓式、确定式、医疗/法律/投资专业结论。
