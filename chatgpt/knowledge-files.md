# ChatGPT Knowledge Files

Custom GPT 上传 Knowledge 时，优先选择下列文件。不要上传原始 PDF 全文，除非你拥有相应授权并确认用途合规。

## 必传

- `README.md`
- `shared/contracts/qz_structured_data_contract.md`
- `shared/contracts/qz_core_report_contract.md`
- `shared/contracts/qz_topic_report_contract.md`
- `shared/prompts/standard_answer_formats.md`
- `shared/prompts/cautious_language_templates.md`
- `shared/references/course_index_full.md`
- `shared/references/glossary_full.md`
- `shared/references/core_concepts_full.md`
- `shared/references/chart_structure_full.md`
- `shared/references/wuxing_full.md`
- `shared/references/topic_index_full.md`

## 按需上传

- 学习问答：`chatgpt/skills/qz-study/SKILL.md` 与 `chatgpt/skills/qz-study/resources/`
- 读盘：`chatgpt/skills/qz-reader/SKILL.md` 与 `chatgpt/skills/qz-reader/resources/`
- 原局核心分析：`chatgpt/skills/qz-core/SKILL.md` 与 `chatgpt/skills/qz-core/resources/`
- 专题分析：对应专题 Skill 的 `SKILL.md` 和 `resources/`

## 不建议上传

- `build/pdf_text/page_*.txt`：这是本地提取中间产物，包含逐页原文文本，不建议作为可分发知识库。
- 原始 PDF：版权归原权利人所有，本项目不重新分发 PDF 原文。
