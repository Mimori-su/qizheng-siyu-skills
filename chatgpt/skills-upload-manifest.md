# ChatGPT Skills Upload Manifest

逐个上传 Skill 时，建议顺序如下：

| 顺序 | Skill | 作用 | 是否核心 |
|---|---|---|---|
| 1 | qz-study | 教材学习、术语解释、学习路线 | 是 |
| 2 | qz-reader | 读盘、提取、校验、结构化数据 | 是 |
| 3 | qz-core | 原局核心分析 | 是 |
| 4 | qz-report | 报告整合 | 是 |
| 5 | qz-love | 感情、婚姻、合盘 | 按需 |
| 6 | qz-wealth | 财运、资产、田宅 | 按需 |
| 7 | qz-health | 健康与疾病取象 | 按需 |
| 8 | qz-career | 事业与官禄 | 按需 |
| 9 | qz-children | 子女与亲子 | 按需 |
| 10 | qz-lawsuit | 官非与法律风险取象 | 按需 |
| 11 | qz-election | 择日 | 按需 |
| 12 | qz-rectifier | 生时校正 | 按需 |

## 上传后自检

- 询问“七政四余是什么”，应触发 qz-study。
- 询问“请读这个盘面文本”，应触发 qz-reader 且不解读吉凶。
- 询问“基于结构化数据做原局分析”，应触发 qz-core。
- 询问“感情/财运/健康专题”，应先要求 Reader/Core 上游结果。
