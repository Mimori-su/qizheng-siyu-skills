# Core 报告规则

Core 是专题 Skill 的上游，报告要便于继承和复核。

## 必备顺序

1. 数据完整性复核：说明哪些字段可用，哪些字段低可信度。
2. 太极与命身：确认命宫、命度、命度主、身宫、身度、身度主和性别取法。
3. 原局结构：宫、度、星、神煞分层，不以单星定论。
4. 五行体用喜忌：说明为什么喜、为什么忌，不能机械套生克。
5. 运势引动：先看原局有没有信息，再看流年、限度、倒限是否触发。
6. 可继承结论：写成专题可复核的证据点，而不是绝对断语。

## 正反双审

每个核心判断必须列支持因素、制约因素、缺失数据和不确定项。若支持证据少于两层，应降级为“可观察信号”。

## 禁止写法

不写恐吓式、确定式、医疗/法律/投资专业结论。
