# 核心概念整理

## 太极

来源页码：3-4, 181-190

### 教材摘要
太极是分析对象的中心。论命主、父母、感情、财运都要先确定太极。

### 核心要点
先定对象，再看围绕对象的宫、度、星、神煞。

### 分析用途
所有专题的分析入口；太极变了，喜忌和取象也可能变。

### 易错点
把命主、命宫主、身宫主混用，或在专题中不说明太极转换。

### 可用于哪些 Skill
qz-core / qz-love / qz-wealth / qz-career

## 阴阳

来源页码：3-8, 344

### 教材摘要
阴阳是喜忌、生克、正偏和转化的底层语言。

### 核心要点
生不一定吉，克不一定凶；喜忌随太极和用处变化。

### 分析用途
解释支持因素和制约因素为什么可以并存。

### 易错点
机械说“生就是吉、克就是凶”。

### 可用于哪些 Skill
qz-study / qz-core

## 三才

来源页码：4-5, 218-227

### 教材摘要
二十八宿为天，十二宫为地，命主为人。

### 核心要点
论宫当论宫，论度当论度。

### 分析用途
Reader 提取宫度，Core 分层分析。

### 易错点
把宫位、度数、星曜混为一个层级。

### 可用于哪些 Skill
qz-reader / qz-core

## 四时

来源页码：5, 36-40

### 教材摘要
四时即月令节气，是判断五行状态的重要依据。

### 核心要点
春木、夏火、秋金、冬水、四季土，并看杂气。

### 分析用途
判断得令、气数和用神可用性。

### 易错点
按农历初一机械换月；教材提示以节气换月。

### 可用于哪些 Skill
qz-study / qz-core

## 五行

来源页码：30-40, 96-148

### 教材摘要
五行是运行变化与取象系统，不只是生克公式。

### 核心要点
要结合本性、时令、太极、生活画面理解。

### 分析用途
确定命主本性如何被成全或受阻。

### 易错点
只背“木克土”而忽略“土培木”等太极转换。

### 可用于哪些 Skill
qz-study / qz-core

## 体用

来源页码：6, 218

### 教材摘要
体是被分析的太极点，用是可围绕太极起作用之物。

### 核心要点
先明体，再论能否为我所用。

### 分析用途
用神和专题取象的基础。

### 易错点
不定体就谈用神。

### 可用于哪些 Skill
qz-core

## 喜忌

来源页码：6, 175-180

### 教材摘要
喜忌围绕尽三和第一个关键星展开。

### 核心要点
第一个星为喜则后面护喜；第一个星为忌则后面制忌。

### 分析用途
形成 Core 报告的支持/制约框架。

### 易错点
不给理由地标喜忌。

### 可用于哪些 Skill
qz-core / topic skills

## 气数

来源页码：7, 38-40

### 教材摘要
教材以水火说明天地流行规律。

### 核心要点
辰到酉重水润，戌到卯重火暖；未申重水，丑寅重火。

### 分析用途
用于判断成局基础和偏枯调候。

### 易错点
有气数就断富贵；教材提醒得气数不一定富贵。

### 可用于哪些 Skill
qz-core

## 女命诀

来源页码：7, 16

### 教材摘要
男命以命度主为命元，女命以身度主为命元。

### 核心要点
Reader 必须记录性别和取法。

### 分析用途
命身判断和生时校正。

### 易错点
男女同盘完全同看，不记录取法。

### 可用于哪些 Skill
qz-reader / qz-core

## 原局与运势

来源页码：7, 149-162

### 教材摘要
原局是信息库，运势是引动。

### 核心要点
先确认原局是否有信号，再看限度流年是否触发。

### 分析用途
所有专题时机判断的基础。

### 易错点
没有原局信号，只凭流年下定论。

### 可用于哪些 Skill
qz-core / topic skills

## 五行不单单是生克

来源页码：8, 30-40

### 教材摘要
教材明确提醒不要过度依赖生克。

### 核心要点
同一五行关系在不同太极下可有不同取象。

### 分析用途
帮助报告写出制约证据而非单点结论。

### 易错点
机械化、公式化断盘。

### 可用于哪些 Skill
qz-study / qz-core

## 排盘软件注意事项

来源页码：16-18, 341

### 教材摘要
教材提到原局盘、流年盘、神煞层、星体分布和软件择日功能。

### 核心要点
Reader 应区分原局盘与流年盘，必要时让用户关闭流年层。

### 分析用途
读盘 OCR 与截图识别。

### 易错点
把蓝色流年星当原局星。

### 可用于哪些 Skill
qz-reader / qz-election
