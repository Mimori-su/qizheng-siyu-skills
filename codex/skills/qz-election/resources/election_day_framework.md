# 择日框架

来源页码：第五十八课 341-344；干支刑冲合害参考 23-26、344。

## 输入要求

- 事项类型：开业、签约、搬家、出行、婚礼、手术、动工等。
- 候选日期与时间范围。
- 现实约束：地点、人员可用性、合同期限、天气、安全、法律要求。
- 是否需要结合命盘：若涉及个人长期事项，建议先有 qz-reader 和 qz-core。

## 观察顺序

1. 明确事项太极：为谁择日、为何事择日。
2. 排除现实不可行日期：安全、合规、身体状况优先。
3. 看太岁、刑冲破害、三合六合、空亡等基础关系。
4. 看事项相关宫星和吉神/大煞。
5. 列出支持因素、制约因素和替代候选。

## 输出边界

择日只能作为传统时间取象参考，不保证某日必吉，也不能替代医疗、法律、工程、金融安全判断。
