# 完整报告章节

`qz-report` 只整合既有报告，不新增断语。

## 推荐顺序

1. 报告声明：传统术数取象参考，非医学/法律/财务/心理专业意见。
2. 数据基础：列出 `qz_structured_data.md`、`qz_core_report.md` 和专题报告来源。
3. 数据完整性：合并各报告的缺失字段和低可信度项。
4. Core 摘要：太极、命身、原局结构、喜忌、可继承结论。
5. 专题章节：感情、财运、健康、事业、子女、官非、择日、校时等。
6. 正反双审：总支持因素、总制约因素和不确定项。
7. 后续确认清单：需要补充的盘面、事件、现实专业意见。

## 合并原则

同一结论重复出现时保留最完整证据；结论冲突时不强行调和，列为不确定项。
