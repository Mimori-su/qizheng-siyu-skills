# 事件映射规则

事件映射用于比较候选时辰，不用于事后制造确定性。

## 映射步骤

1. 事件分类：家庭、学业、事业、迁移、感情、健康、官非、财务、子女。
2. 找主题宫位：例如感情看夫妻/男女/迁移，事业看官禄/福德/财帛，健康看疾病宫。
3. 找命身差异：候选时辰是否导致命宫、命度、身宫、身度或度主变化。
4. 找运势引动：流年、限度、倒限是否触发原局信号。
5. 记录支持与冲突：不能只保留符合候选时辰的证据。

## 证据等级

- 强：原局主题明显，且事件年份被运势触发。
- 中：原局主题可见，但时间触发不清。
- 弱：只有单一神煞或模糊取象。
- 无效：事件时间不明、数据来源低可信度、与候选差异无关。
