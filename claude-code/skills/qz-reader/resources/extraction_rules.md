# 提取规则

来源页码：第三课 16-22；排盘软件注意事项见核心概念整理。

## 输入优先级

1. 用户粘贴的盘面文本：优先级最高，通常字段可复制，错误主要来自用户删改。
2. PDF 文本层：适合命盘 PDF 和教材页级复核，必须保留页码或文件名。
3. 截图 OCR：只作为初稿，中文星宿、宫位和度数都要标为中低可信度。
4. 用户手工补充：可补缺，但要标注“用户输入”来源。

## 必提字段

- 元信息：出生日期、出生时间、出生地点、性别、排盘软件、数据来源、读取方式。
- 命身：命宫、命度、命度主、身宫、身度、身度主、男命/女命取法。
- 星曜：日、月、金、木、水、火、土、罗、气、计、孛的位置、宫位、星宿/度、状态。
- 结构：十二宫、二十八宿、神煞、流年、限度、倒限。

## 记录要求

- 每个字段都要有“来源”和“可信度”。
- 原局、流年、限度必须分开标注，不能把外层流年误写成原局。
- OCR 中相似字要列为可疑项，例如“氐/氏”“房/昴”“罗/络”“计/讣”。
- Reader 只做数据标记，不判断吉凶、不分析感情财运疾病。
