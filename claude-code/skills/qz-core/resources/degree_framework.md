# 度与穿度框架

来源页码：星盘组成 16-22；度与穿度相关说明 34-35、82、218-227。

## 基本分层

- 宫：十二宫，偏领域、环境、社会关系，可理解为“地”。
- 度：二十八宿与度主，偏命元、星宿气质和天层结构，可理解为“天”。
- 星：七政四余落在宫度中，形成具体作用点。
- 神煞：辅助取象，不可越过宫度星的主体结构。

## 命身字段

- 命宫：命的宫位太极，不等于全部命元。
- 命度：立命所在星宿之度，是 Core 必看字段。
- 命度主：教材强调的命主核心字段。
- 身宫/身度/身度主：现实承载、身体感应和女命取法中尤其关键。

## 尽三之后看度

教材在尽三案例后说明：前面尽三全部是论宫组合，接下来论度，全部看四度相通。实务顺序是：

1. 尽三三步先论宫，得出用神、喜忌和组合成败。
2. 接着看命度。
3. 再看命主星所在的度。
4. 再看身度。
5. 度层若有同度、穿度、四度相通或同度相犯，会调整尽三结论：尽三败而度救，可有补救；尽三成而度败，则层次和应象会下落。

## 同度与同宫

- 尽三阶段主要论宫组合，同宫/三方四正感应为主。
- 论度阶段才重点看同度、穿度和四度相通。
- 余奴相犯也要分层：论尽三时看宫是否感应，论度时看同度是否感应。

## 易错点

- 用命宫主替代命度主。
- 只看宫不看度，或只看度不看落宫环境。
- 未区分命度、身度、命主星所在度。
- Reader 未记录来源可信度，Core 直接断语。
