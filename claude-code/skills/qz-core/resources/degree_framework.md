# 度与穿度框架

来源页码：星盘组成 16-22；度与穿度相关说明 34-35、82、218-227。

## 基本分层

- 宫：十二宫，偏领域、环境、社会关系，可理解为“地”。
- 度：二十八宿与度主，偏命元、星宿气质和天层结构，可理解为“天”。
- 星：七政四余落在宫度中，形成具体作用点。
- 神煞：辅助取象，不可越过宫度星的主体结构。

## 命身字段

- 命宫：命的宫位太极，不等于全部命元。
- 命度：立命所在星宿之度，是 Core 必看字段。
- 命度主：教材强调的命主核心字段。
- 身宫/身度/身度主：现实承载、身体感应和女命取法中尤其关键。

## 尽三只论宫（三步起手）

尽三是看盘起手，只论宫不论度，从月令和本命之性出发，三步而止，每步只看“本宫及三方四正”：

1. 第一步：命宫及其对宫、三方四正；先入宫的星先论，命宫本星先忽略。
2. 第二步：命主星所在宫及三方四正；注意命主星先遇到哪颗星，命主星本身先忽略。
3. 第三步：月亮（身）所在宫及三方四正；月亮为身、为归结，本身先忽略。

教材原文：“尽三看论宫，身命度看论度。”所以三步只在“宫”这一层定喜忌骨架和用神，尽三里不看度。

## 尽三之后：三度论度断

尽三论宫之后才论度，原则是“论宫当论宫，论度当论度”，两条线不混看。论度按三度，每度都看“四度相通”——二十八宿中同一五行的四个度互通——上有什么星：

1. 命度：看命度所属五行的四个度上的星。
2. 命主星所在的度：看该度所属五行的四个度上的星。
3. 身度：看身度所属五行的四个度上的星。

度是“天”，会去调整尽三（地、人）的结论：尽三败而度救，可有补救；尽三成而度败，层次和应象会下落。

## 余奴相犯：宫、度两条线分开看

余奴犯主指正星与对应余星相见：罗与火、孛与水、气与木、计与土；是否为患看命对该物的喜忌。

- 论宫相犯：尽三阶段三方四正见到即相犯（对宫相犯、三方相犯；就对相犯而言三方不管四正）。
- 论度相犯（同度相犯）：论度阶段，余奴对子落在同一五行的度上（四度相通之内）即相犯，例如土在角木、计在井木（皆木度），或火在星日、罗在房日（皆日度）。

犯死与犯不尽：

- 犯死：没有第三颗星在场才算犯死，犯死后不参与尽三生克。
- 犯不尽：四度相通或同宫三方上还有第三星、有日月在场、或有生克条件，就犯不尽，仍参与生克。
- 即使犯死不参与生克，星体气场仍在，仍影响层次与取象，不能当作完全消失。

## 易错点

- 把尽三拿去论度，或把同度相犯的度规则提前用进尽三论宫。
- 把“同度/四度相通”误解为必须精确同一度或同宫；它指同一五行的四个度互通。
- 用命宫主替代命度主。
- 只看宫不看度，或只看度不看落宫环境。
- 未区分命度、身度、命主星所在度。
- Reader 未记录来源可信度，Core 直接断语。
