# 五行命型断盘要点（木火土月日金水）

本表用于 qz-core 断盘起手：定太极命身后，先按命型确认本命之性与命主星旺衰，再据此成全本性、判断喜忌与尽三成败。整理自第十七至二十三课，是断盘必用的本性框架，不替代原书；遇教材未明确处直接说明。页码为 PDF 页码范围。

## 断盘起手如何用

1. 定太极命身后，先按命型读出本命之性（要成全什么）。
2. 衡量命主星旺衰：得令、升殿、入垣，得一不弱，三得很旺，三不得很弱；人命宜旺不宜衰，太旺太弱都是偏枯。
3. 结合气数（辰至酉月喜水、戌至卯月喜火）判断调候是否到位；得气数不一定富贵，但要富贵必得气数。
4. 带着“如何成全本性”进入尽三，再以三度论度复核；五行先用不先主，论用终是在论命。
5. 同组合先后不同则象不同，要分谁得天时、有情无情；柔不等于弱。

来源页码：96-148。

## 木命

- 本性：生长生发；要分辨活木/苍松/栋梁与烂木/麦梗/死木，各有用处。
- 四时：春木旺，最怕金伤根；夏木花艳但怕南奔、火焚（虚而无水易偏枯出凶）；秋木弱，但木旺逢金可金木相成成栋梁；冬木归藏需护根，亦有梅花不畏寒。
- 喜忌：一般喜土培（水中植物例外）、喜向阳；木见气（忌）再见金、木见金（伤根）再见气，都是秋木胜春木，核心在谁得天时。
- 断盘提醒：用印而木已旺时不喜再见比劫（如罗），否则一下落为普通（身旺比劫重）。

来源页码：96-100。

## 火命

- 本性：发光发热炎上；怕被灭，也怕自焚（过旺成爆火）。
- 四时：夏旺；春不旺不弱（木旺暗生火）；秋弱（金多火炽）；冬弱（水旺暗克）。
- 喜忌：火性本虚，喜借木为体（第一星逢木多喜，冬火尤甚）；喜水相济（辰至酉月尤喜）；怕土晦（无光，易出暗昧之事）；喜火金同明、喜见日（火见日为朝君）。
- 断盘提醒：万家灯火（见木成或火金同明）对烽火戏诸侯（见木又见金伤、财坏印）；火炎土燥（夏秋火见木见土无水）主凶顽、多学少成、上班象。

来源页码：101-106。

## 土命

- 本性：包容养育、厚重、主信；成局有母爱包容，破局则刻薄自私、虚言无信。
- 四时：辰戌丑未旺；春土虚（喜火生，怕见木杀承载太重、怕水寒成混泥）；夏土怕火炎土燥必水济；秋土归实（用官杀更贵、喜金更富，土金富实）；冬土怕冻必火暖。
- 喜忌：夏土先水最利财；先木（杀）再水再火可以印化杀、寒门贵子，财官印皆为我所用。
- 断盘提醒：星多、余奴多、交战多易杂，杂中可求小成；女命官杀混杂则感情难顺；土命忌官杀者易被纠缠榨干。

来源页码：107-112。

## 月命

- 本性：月为母后，本无光，全在借日而明、借火而光、借木清辉；最容易成命（可圆可缺）。
- 喜忌：喜日金火同明、喜木月清辉；忌土计晦月（易暗昧之事）；不太怕燥（月是水精）但怕寒，最忌寒月单行（主孤主贫）；喜行日限。
- 地利：喜未酉等阴地、本垣未宫、井木与四水度；忌四日度（阴阳失位）、四土度。满月（十五十六）我最大，反怕争光，满月不如残月。
- 断盘提醒：日月命不论十神，以宫为重，必分昼夜。子嗣口诀：昼生太阴独在男女宫无辅则无子；夜生唯水辅日在男女宫反主有子。

来源页码：113-118。

## 日命

- 本性：君王照耀天下；喜金水辅阳、木气伴日（日为用神时忌木气蔽日）；忌土计晦日（单土为白云、土计为乌云）；最怕火罗犯日，尤怕罗（罗如十日，是太阳比劫）。
- 特性：太阳是最大贵人；四日度有细差（星日怕罗气、昴日带金意怕火罗、虚日怕计气、房日怕罗孛）；日命不论十神，以宫为重，必分昼夜；论日命如看天气，晴则好、乌云暴雨则不好。
- 人生底色：成命难、孤独多挫（寡人），有病方为贵。
- 断盘提醒：金孛穿度且金为男女主可见偷生、出轨怀孕打胎象；倒限是寻最忌之星，命、限、流全引动即寿命关口。

来源页码：119-130。

## 金命

- 本性：丽质高贵精致（金光闪闪、宝剑珠宝），成命多帅哥美女、多才多艺（金本身即艺术）。
- 四时：春金虚（喜土生、火暖、木相成，怕金寒水冷尤寅卯）；夏金喜火炼须一水一火（金脆火炎多夭折），喜水金白水清；秋金已成怕炼过、喜水、金土相成主富；冬金最怕金寒水冷（沉冤牢狱、又孤又贫）。
- 喜忌：金煞气重，最怕带刃、天雄地雄（金星为杀更逢煞，非命兼王法）；金水孛主情欲桃花，金命女多感情难顺；金火同病都怕土晦（金对土是印、火对土是食伤）。
- 断盘提醒：身弱不受生，尽三再完美也可能普通人；财多身弱用食伤为才偏性执；金水伤官最聪明（金无余奴，最多两制）。

来源页码：131-141。

## 水命

- 本性：之方则方、之圆则圆，善变喜流、怕干涸；成全为流（不涸）、清（不浊）、静（不波涛），喜源头、喜河道宽广。
- 四时：春水怕土混（成混泥）；夏水来火不怕（水火相济）；秋水融融最怕土混（易被奸）；冬水必火（无火无生机，全见火怕冰崩）。
- 喜忌：喜木为水木清华（大才子高材生；女命水木清华成易情伤克夫）；喜火相济。见木为喜是有河道可灌溉；见木喜又逢金伤木则河道坏成洪水（伤人）。
- 断盘提醒：用神只在尽三定。尽三无用神时命身度穿木、土皆喜；尽三若已会到土（土为用神），木穿度即伤用。

来源页码：142-148。

## 共性断盘提醒

- 木命：先解决气数（暖或润）前提，再论生克，否则被焚或漂流。
- 火命与金命：都怕土晦，但火对土为食伤、金对土为印，晦的象不同。
- 日命与月命：都不论十神、以宫为重、必分昼夜；太阳怕争光，月亮平时不怕争光但满月怕。
- 土命与水命：土怕崩怕燥、水怕混怕涸，都要先解决调候再论生克。
- 度只调整尽三结论：度判断有没有护住尽三定的用神，度里不另算生克（见 degree_framework.md）。

来源页码：96-148。
