# Core 报告规则

Core 是专题 Skill 的上游，报告要便于继承和复核。

## 必备顺序

1. 数据完整性复核：说明哪些字段可用，哪些字段低可信度。
2. 太极与命身：确认命宫、命度、命度主、身宫、身度、身度主和性别取法。
3. 尽三起手：按命主太极和命身取法，列出尽三第一步、第二步、第三步，先定喜忌骨架。
4. 度层复核：尽三论宫之后，立即看命度、命主星所在度、身度等三度；按四度相通和同度感应复核尽三成败。
5. 原局结构：在尽三和度层骨架下看星曜、神煞和宫位专题，不以单星定论。
6. 五行体用喜忌：说明为什么喜、为什么忌，不能机械套生克。
7. 运势引动：先看原局有没有信息，再看流年、限度、倒限是否触发。
8. 可继承结论：写成专题可复核的证据点，而不是绝对断语。

## 尽三位置

尽三不是 Reader 的职责，也不是专题 Skill 可跳过的步骤。它属于 qz-core 的正式解盘起手：Reader 先把命身、宫度、星曜提取完整；Core 定太极后立即进入尽三。教材明确提示“前面尽三全部是论宫组合，那么接下来论度”，所以尽三之后要紧接命度、命主星所在度、身度等度层；度层用于调整、成全或破坏尽三结论，然后才继续看星曜神煞、五行体用和运势。

## 正反双审

每个核心判断必须列支持因素、制约因素、缺失数据和不确定项。若支持证据少于两层，应降级为“可观察信号”。

## 禁止写法

不写恐吓式、确定式、医疗/法律/投资专业结论。
