# 财运专题框架

来源页码：第四十五至四十七课 278-290；喜忌取法参考 175-180。

## 分析前置

- 必须先读取 `qz_structured_data.md` 和 `qz_core_report.md`。
- 若 Core 没有给出太极、命身、喜忌和原局层次，先回退到 qz-core。
- 财运专题只谈传统术数里的资源结构、取得方式、守成压力和运势引动。

## 关键观察点

- 财帛宫：收入、现金流、可动用资源。
- 田宅宫：不动产、积累、家宅与资产承载。
- 官禄宫：事业平台、名分、制度资源，与财的取得路径有关。
- 福德宫：享用、稳定度、长期余裕。
- 命身与用神：财是否为我所用，是否护身或反成压力。
- 运势：原局已有财象时，再看流年、限度是否引动。

## 正反双审

- 支持因素：财宫有用、财星得地、原局有承载、运势引动。
- 制约因素：财象被破、身弱难任、用忌不清、只有单一神煞。
- 不确定项：缺流年、缺田宅/财帛宫信息、OCR 低可信度。
