# 财运报告规则

财运报告必须继承 Core，不重新读盘、不独立定喜忌。

## 必备章节

- 问题与数据基础：列出用户问的是收入、资产、田宅、破财还是投资。
- 教材依据：标注财运专题 278-290、基本喜忌 175-180 等页码。
- 支持因素：至少列 2 类证据，不能只写单星或单宫。
- 制约因素：必须列出反向证据或数据不足。
- 谨慎结论：只写趋势、结构和注意点，不写确定收益。

## 表达边界

- “财象较活跃”优于“必发财”。
- “守财压力需要结合运势复核”优于“必破财”。
- “投资议题请交给现实财务评估”必须出现在投资相关报告里。

## 输出建议

可给现实层面的记录预算、风险分散、学习技能、核对合同等通用建议，但不得推荐具体交易标的或收益率。
