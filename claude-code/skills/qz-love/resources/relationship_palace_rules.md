# 感情宫位规则

## 夫妻宫

夫妻宫偏长期关系、伴侣位置、关系制度和共同生活结构。它是重要观察点，但不能单宫决定婚姻结果。

## 男女宫

男女宫可看男女互动、情感表达、子女相关延伸取象。涉及生育时要转交 qz-children，并提醒医学问题咨询医生。

## 迁移宫

迁移宫可看外缘、异地、外部环境、关系中的距离与变化。感情专题中常用于判断关系是否受外部因素影响。

## 合参规则

- 宫位要合命身、命度主/身度主、相关星曜和神煞。
- 桃花、孛星等只作辅助，不作唯一依据。
- 运势只能说明引动可能，不能保证具体事件。

## 正反双审

每个关系判断都要列支持因素和制约因素；若制约因素不足，主动寻找反证。
