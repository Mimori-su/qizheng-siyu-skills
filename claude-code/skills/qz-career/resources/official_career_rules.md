# 官禄与事业规则

官禄不是单纯“当官”，在现代报告中可对应职位、组织规则、职业名望、责任压力和评价体系。

## 官禄取象

- 正向：职位平台、制度资源、专业身份、公开评价、领导支持。
- 中性：责任、约束、规则、绩效、审查。
- 负向：压力、规制冲突、上级不合、名誉受损。

## 与其他宫位合看

- 官禄 + 财帛：职业是否能转化为收入。
- 官禄 + 福德：工作状态与长期承受力。
- 官禄 + 迁移：外部平台、客户、异地发展。
- 官禄 + 奴仆：团队、同事、下属、合作方。

## 分流规则

- 事业发展留在 qz-career。
- 合同、诉讼、处罚、仲裁等现实法律风险交给 qz-lawsuit，并提醒咨询律师。
