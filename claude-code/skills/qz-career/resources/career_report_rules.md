# 事业报告规则

## 必备结构

- 用户问题：求职、升职、转行、领导关系、名望还是职业方向。
- 数据基础：列出已继承的 Core 结论和缺失字段。
- 支持因素：官禄、命身、用神、财帛、迁移中的正向证据。
- 制约因素：反向证据、时机不足、数据缺口、现实不确定。
- 谨慎结论：写职业倾向和准备方向，不写“必升”“必败”。

## 报告边界

- 不能保证录用、升职、考编、签约或创业成功。
- 不能替代职业咨询、劳动法律咨询或心理咨询。
- 对涉及合同纠纷、仲裁、诉讼的问题，应交给 qz-lawsuit 并建议咨询律师。

## 可行动建议

建议应落到现实行为：梳理能力证据、补技能、沟通预期、管理风险、核对合同、争取反馈。
